# Fatal Labyrinth Clone (Haxe + Heaps)

A roguelike dungeon crawler inspired by Fatal Labyrinth.

## Requirements
- Haxe 4.3+
- HashLink
- Heaps.io

## Setup
```bash
haxelib install heaps
haxelib install hlsdl
```

## Build & Run
```bash
haxe compile.hxml
hl bin/game.hl
```

## Controls
- **Arrows**: Move / Attack
- **Enter**: Select in Menu
- **ESC**: Pause / Back
